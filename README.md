# guandan-score
掼蛋计分器 - 3人升级制打牌记录工具

你将可以通过 https://lapbobo.github.io/guandan-score/ 访问

guandan-score 是一个基于网页的掼蛋（三人升级制）纸牌游戏计分工具。

该项目提供了一个简洁的计分记录应用程序，专门用于追踪3人升级制掼蛋游戏中各玩家的积分。

项目支持PWA（渐进式网络应用）部署，可通过GitHub Pages直接访问使用。

最新版本修复了庄家为"甜大官"（ID=0）时的显示bug。

主要功能点
3人升级制掼蛋游戏计分记录
实时积分计算与显示
庄家身份标记与管理
PWA应用支持（可离线使用）
轻量级网页应用
技术栈
HTML（前端页面）
JavaScript（业务逻辑）
PWA Manifest.json（应用配置）
GitHub Pages（部署平台）
